📚 Bookstore Management System – Full Stack Web App
📌 Project Overview

The Bookstore Management System is a full-stack web application designed to help bookstore owners manage books, users, and orders efficiently. It provides a seamless interface for customers to browse and purchase books, while admins can manage inventory and orders.

🎯 Objective

To build a scalable web application using:

Frontend: React
Backend: Node.js + Express
Database: MongoDB

The system includes authentication, role-based access, inventory management, and order processing.

🚀 Features
👤 User Features
Register and login (JWT authentication)
Browse books with search & filters
View book details
Add to cart and checkout
View order history


🛠️ Admin Features
Add, update, delete books
Manage orders
Update order status
Access admin dashboard
